This fallback page is only used when the Capacitor shell is synced without a CAPACITOR_SERVER_URL.

For live haptics testing against the deployed Next.js app, run the sync command with a remote URL, then open the native project in Xcode or Android Studio.

  • npm run capacitor:sync:live -- https://williamtemple.app
  • npm run capacitor:open:ios or npm run capacitor:open:android
